## 项目结构

```
├── index.html  # 主页
├── README.md   # 项目说明
├── style.css   # 主页样式
├── nav.js      # 主页脚本
├── img         # 图片目录
│   ├── wechat.jpg  # 微信图片
│   ├── avatar.jpg  # 头像
└── articles       # 文章目录
    ├── 1.html  # 文章1
    ├── 2.html  # 文章2
    ├── 3.html  # 文章3
    ├── 4.html  # 文章4
```
